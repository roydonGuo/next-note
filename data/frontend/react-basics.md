# React 基础

## 什么是 React？

React 是一个用于构建用户界面的 JavaScript 库。它由 Facebook 开发，现在由社区维护。

## 核心概念

### 组件
React 应用由组件组成。组件是独立的、可重用的代码片段。

### JSX
JSX 是 JavaScript 的语法扩展，允许我们在 JavaScript 中编写类似 HTML 的代码。

### Props
Props 是传递给组件的参数，类似于函数的参数。

### State
State 是组件内部的数据，可以随时间变化。

## 示例代码

```jsx
function Welcome(props) {
  return <h1>Hello, {props.name}!</h1>;
}
```

## 总结

React 让构建交互式用户界面变得简单和高效。
